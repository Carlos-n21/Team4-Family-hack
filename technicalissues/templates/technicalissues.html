Mobile Devices

Get help with your smart phones and tablets

'How to' guide

Troubleshooting

Connect to wifi

  • Open the Settings app.
  • Tap Wi-Fi.
  • Make sure the Wi-Fi is turned on.
  • You should see a list of available networks. Tap the one you want to join.
  • Enter the password if asked, and then hit Connect.

Change screen brightness

  • Swipe down from the top of your screen.
  • You should see a little brightness slider - move it left or right to make your screen darker or brighter.
  • Or, you can go to Settings > Display and adjust it there.

Set an alarm

  • Open the Clock app.
  • Tap the Alarm tab.
  • Tap the + icon to add a new alarm.
  • Set the time and any other preferences, then tap Save.

Sound not working

  • Check the volume settings to ensure it's turned up.
  • Test with a different app to rule out the possibility of app-related issues.
  • If you are using headphones, make sure they are plugged in properly.
  • If the issue persists, try restarting your phone or resetting sound settings.

Bluetooth not connecting

  • Make sure Bluetooth is turned on in your Settings.
  • Ensure the device you are connecting to is also in a pairing mode or discoverable.
  • Try 'forgetting' the device in your Bluetooth settings and reconnecting.
  • If the issue persists, restart both devices and try again.

Mobile App not working properly

  • First, try closing and restarting the app.
  • If that doesn't help, check for updates for the app in the app store.
  • If it is still not working, try restarting your phone.
  • As a last resort, clear the app's cache or reinstall it.

Laptops/Desktops

Get help with your laptops and desktops

'How to' guide

Troubleshooting

Connect to wifi

  • Check the router connection.
  • Restart by Unplugging.
  • Restart your computer.
  • Check Wi-Fi Settings. Windows: Use the Wi-Fi icon to confirm you're connected.
    Mac: Use the top-right Wi-Fi icon to join your network.
  • Forget and Reconnect
    Windows: Settings > Network > Manage Known Networks > Forget and reconnect.
    Mac: System Preferences > Network > Remove and rejoin Wi-Fi.
  • Update Wi-Fi Drivers
    Windows: Device Manager > Network Adapters > Update driver.
    Mac: Check for system updates in Apple Menu > Software Update.

Computer won’t turn on

  • Check the Power Source
    Laptop: Make sure your laptop is plugged in, and check if the charging light is on. It may take a few minutes to show signs of life if completely drained.
    Desktop: Ensure the power cable is connected to both the wall and the power supply unit.
  • Try a Hard Reset
    Laptop: Hold the power button for 10-15 seconds, then press it again.
    Desktop: Unplug from power for 30 seconds, plug back in, then power on.
  • Check the Display
    Laptop: Adjust brightness or check for screen damage.
    Desktop: Ensure the monitor is correctly connected and powered on.

Computer is running slow

  • Restart the Computer
    Close all programs and restart to clear temporary files.
  • Close Unnecessary Programs
    Windows: Ctrl + Shift + Esc to open Task Manager.
    Mac: Cmd + Option + Esc to open Force Quit and close unused apps.
  • Free Up Disk Space
    Windows: Settings > System > Storage > Disk Cleanup.
    Mac: Finder > About This Mac > Storage. Remove unused files/apps.
  • Check for Malware
    Windows: Use Windows Defender or a trusted antivirus.
    Mac: Use a Mac-compatible antivirus tool.

Computer is not responding (Frozen)

  • Try a Keyboard Shortcut to Close the Program
    Windows: Press Ctrl + Alt + Delete to open Task Manager and select End Task for any frozen program.
    Mac: Press Cmd + Option + Esc to open Force Quit, then close the unresponsive app.
  • Restart the Computer
    If the above doesn't work, press and hold the power button for 10-15 seconds to force a shutdown, then turn the computer back on.
  • Check for Software Updates
    Windows: Go to Settings > Update & Security > Windows Update > Check for updates.
    Mac: Apple Menu > System Preferences > Software Update.

Keyboard or Mouse Not Working

  • Check the Connection
    Laptop: If using an external mouse or keyboard, ensure they are plugged in properly.
    Desktop: If using a wireless keyboard/mouse, check the batteries or reconnect the USB receiver.
  • Restart the Computer
    Sometimes restarting can help fix minor issues with input devices.
  • Check Keyboard and Mouse Drivers
    Windows: Go to Device Manager, right-click on Keyboards or Mice and other pointing devices, and select Update Driver.
    Mac: Check System Preferences > Keyboard or Mouse to make sure the settings are correct.
  • Test with Another Device
    Try connecting the keyboard or mouse to another computer to see if the problem is with the device itself.

Computer Display is Too Dim (Brightness)

  • Adjust the Brightness Manually
    Laptop: Use the Fn key + brightness keys (sun symbol) to adjust screen brightness.
    Desktop: Use physical buttons on the monitor to adjust brightness.
  • Check Power Settings
    Windows: Go to Settings > System > Display. Adjust brightness and power settings to prevent dimming.
